# 蓝图：素数定理的解析证明

## 主任务

证明素数定理（Prime Number Theorem）：
\[
\pi(x) \sim \frac{x}{\log x} \quad (x \to \infty)
\]
其中 \(\pi(x)\) 表示不超过 \(x\) 的素数个数。

---

## 次任务分解

### 次任务 1：Chebyshev 估计的初等推导

- **子任务 1.1**：证明 \(\vartheta(x) = \sum_{p \le x} \log p = O(x)\)。
- **子任务 1.2**：证明 \(\psi(x) = \sum_{p^k \le x} \log p = \vartheta(x) + O(\sqrt{x})\)。
- **子任务 1.3**：建立上下界 \(c_1 x \le \psi(x) \le c_2 x\)（例如 \(c_1 = \log 2/2,\; c_2 = \log 2\)）。
- **子任务 1.4**：导出 \(\pi(x) = \Omega(x/\log x)\) 和 \(\pi(x) = O(x/\log x)\)。

### 次任务 2：Dirichlet 级数与 Riemann zeta 函数的基本性质

- **子任务 2.1**：定义 Riemann zeta 函数 \(\zeta(s) = \sum_{n=1}^{\infty} n^{-s}\) 并证明其绝对收敛于 \(\Re(s) > 1\) 的区域。
- **子任务 2.2**：证明 Euler 乘积公式 \(\zeta(s) = \prod_{p} (1 - p^{-s})^{-1}\)，其中 \(p\) 取遍所有素数。
- **子任务 2.3**：解析延拓 \(\zeta(s)\) 到 \(\Re(s) > 0\)，并建立函数方程（陈述即可，不要求全部推导）。
- **子任务 2.4**：证明 \(\zeta(s)\) 在 \(\Re(s) \ge 1\) 上除了 \(s=1\) 处的一阶极点外解析，且 \(\operatorname{Res}_{s=1} \zeta(s) = 1\)。

### 次任务 3：Perron 型公式的建立

- **子任务 3.1**：定义 Von Mangoldt 函数 \(\Lambda(n)\)，并写出 \(\psi(x) = \sum_{n \le x} \Lambda(n)\)。
- **子任务 3.2**：证明 Dirichlet 级数 \(-\frac{\zeta'(s)}{\zeta(s)} = \sum_{n=1}^{\infty} \frac{\Lambda(n)}{n^{s}}\)，对 \(\Re(s) > 1\)。
- **子任务 3.3**：应用 Perron 公式表达 \(\psi(x) = \frac{1}{2\pi i} \int_{c-i\infty}^{c+i\infty} \left( -\frac{\zeta'(s)}{\zeta(s)} \right) \frac{x^s}{s} ds\)（\(c>1\)）。
- **子任务 3.4**：截断积分，给出误差项估计。

### 次任务 4：零区域与 \(\zeta(s)\) 的零点自由区域

- **子任务 4.1**：证明 \(\zeta(s) \neq 0\) 对 \(\Re(s) \ge 1\)（非零区域）。
- **子任务 4.2**：证明更精细的零点自由区域：存在常数 \(C>0\)，使得 \(\zeta(s) \neq 0\) 当 \(\Re(s) > 1 - \frac{C}{\log |\Im(s)|}\)（对充分大的 \(|\Im(s)|\)）。
- **子任务 4.3**：由零点自由区域导出 \(-\frac{\zeta'(s)}{\zeta(s)}\) 在带形区域的解析性及增长控制。
- **子任务 4.4**：估计 \(\left| \frac{1}{2\pi i} \int_{\text{轮廓}} \right|\) 的误差，得到 \(\psi(x) = x + O(x e^{-c\sqrt{\log x}})\)。

### 次任务 5：主项提取与误差项控制

- **子任务 5.1**：将积分路径向左平移至 \(\Re(s) = 1 - \delta(x)\)，其中 \(\delta(x)\) 依赖于零点自由区域。
- **子任务 5.2**：计算 \(s=1\) 处留数贡献 \(x\)。
- **子任务 5.3**：估计水平积分和垂直积分的界，得到 \(\psi(x) = x + O(x \exp(-c\sqrt{\log x}))\)。
- **子任务 5.4**：利用分部求和，从 \(\psi(x)\) 的渐近式导出 \(\pi(x) = \frac{x}{\log x} + O\left(\frac{x}{(\log x)^2}\right)\)。

### 次任务 6：误差项的初等改进（Walfisz 型估计）

- **子任务 6.1**：引入更精细的零点自由区域（如 Vinogradov–Korobov 型区域）。
- **子任务 6.2**：证明 \(\psi(x) = x + O(x \exp(-c (\log x)^{3/5} / (\log \log x)^{1/5}))\)。
- **子任务 6.3**：由积分平移得到相应的素数定理误差项。
- **子任务 6.4**：对比不同误差阶的历史结果，总结最优已知估计。

### 次任务 7：元素法（初等证明）的概要

- **子任务 7.1**：引入 Selberg 公式：\(\sum_{p\le x} \log^2 p + \sum_{pq\le x} \log p \log q = 2x\log x + O(x)\)。
- **子任务 7.2**：通过初等组合方法推导 \(\sum_{n\le x} \Lambda(n) \log n = x\log x - x + O(\log x)\)。
- **子任务 7.3**：利用 Selberg 公式消去 \(\log n\)，得到 \(\psi(x) = x + o(x)\)。
- **子任务 7.4**：总结初等证明与解析证明的异同，以及 Erdős–Selberg 贡献的历史意义。

### 次任务 8：数值验证与计算实例

- **子任务 8.1**：编写代码计算 \(\pi(x)\) 与 \(x/\log x\) 的比值，对 \(x=10^3,10^4,\dots,10^7\)。
- **子任务 8.2**：用图形展示相对误差 \(\frac{\pi(x) - x/\log x}{x/\log x}\)。
- **子任务 8.3**：验证 Chebyshev 上下界的实际宽松程度。
- **子任务 8.4**：测试零点自由区域对误差项的影响（通过近似计算）。

### 次任务 9：推广到算术级数中的素数定理（Dirichlet 型）

- **子任务 9.1**：定义 Dirichlet L 函数 \(L(s,\chi)\)，并陈述其基本性质。
- **子任务 9.2**：证明 L 函数在 \(\Re(s) \ge 1\) 上无零点（对实特征除外）。
- **子任务 9.3**：建立 \(\psi(x;q,a)\) 的显式公式，并利用 L 函数的零点自由区域导出渐近式。
- **子任务 9.4**：证明算术级数中的素数定理：\(\pi(x;q,a) \sim \frac{x}{\varphi(q)\log x}\)，其中 \((a,q)=1\)。

### 次任务 10：现代发展——素数定理的无条件误差项与零点密度猜想

- **子任务 10.1**：介绍 Riemann 假设下的最优误差项 \(\psi(x) = x + O(\sqrt{x} \log^2 x)\)。
- **子任务 10.2**：讨论当前无条件误差项的最佳结果（如 Korobov–Vinogradov 及后续改进）。
- **子任务 10.3**：解释零点密度猜想与误差项之间的联系。
- **子任务 10.4**：总结素数定理在现代数论中的延伸：如函数域中的类比、筛法中的应用等。
